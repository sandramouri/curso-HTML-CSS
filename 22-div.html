<div> en HTML

El elemento HTML es un contenedor genérico.

Como contenedor "puro" el elemento <div> no posee un significado inherente.

Dicho de otra forma. No tiene ningún efecto sobre el contenido o el diseño hasta que se le aplica un estilo de alguna manera usando CSS (por ejemplo, se le aplica un estilo directamente o se aplica algún tipo de modelo de diseño como Flexbox a su elemento principal).

<div>, se utiliza generalmente para:

  • Agrupar contenido de modo que se le pueda aplicar estilo fácilmente usando los atributos class e id.
  • Marcar una sección de un documento como escrita en un idioma diferente (usando el atributo lang).
  • etc...
