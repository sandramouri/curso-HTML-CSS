Prioridades en CSS

Para evitar conflictos y forzar el uso de una propiedad de un selector usar !important en la propiedad

    --- CSS

    CSS interno:

    <head>
      <style>
        ol {
          color: green !important;
        }
      </style>

      <link rel="stylesheet" href="assets/css/43-prioridad-css-3.css" type="text/css">
    </head>

    CSS externo:

    ul {
      color: purple !important;
    }
    
    --- HTML
    <ol style="color: blue;">
      <li>Primero</li>
      <li>Segundo</li>
      <li>Tercero</li>
    </ol>

    <ul style="color: blue;">
      <li>Uno</li>
      <li>Dos</li>
      <li>Tres</li>
    </ol>

  2. Segundo
  3. Tercero
