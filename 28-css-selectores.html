Selectores CSS

Selector universal

Selecciona todos los elementos.

Opcionalmente, puede estar restringido a un espacio de nombre específico o a todos los espacios de nombres.

Sintaxis: * ns|* *|*

Ejemplo: * se aplicará a todos los elementos del documento.

    * {
      font-style: italic;
    }

Selector de tipo

Selecciona todos los elementos que coinciden con el nombre del elemento especificado.

Sintaxis: eltname

Ejemplo: h2 se aplicará a cualquier elemento <h2>.

    h2 {
      font-style: normal;
      color: darkgreen;
    }

    h3 {
      font-style: normal;
      color: crimson;
    }

Selector de clase

Selecciona todos los elementos que tienen el atributo de class especificado.

Sintaxis: .classname

Ejemplo: .index seleccionará cualquier elemento que tenga la clase "index".

Selector de ID

Selecciona un elemento basándose en el valor de su atributo id. Solo puede haber un elemento con un determinado ID dentro de un documento.

Sintaxis: #idname

Ejemplo: #toc se aplicará a cualquier elemento que tenga el ID "toc".

Título H3 sin CSS

Título H3 con CSS aplicado por selector de ID

    --- CSS

    #tit3 {
      color: slateblue;
    }

    --- HTML
    <h3 id="tit3">Título H3 con CSS aplicado por selector de ID</h3>

Selector de atributo

Selecciona elementos basándose en el valor de un determinado atributo.

Sintaxis: [attr] [attr=value] [attr~=value] [attr|=value] [attr^=value] [attr$=value] [attr*=value]

Ejemplo: [autoplay] seleccionará todos los elementos que tengan el atributo "autoplay" establecido (a cualquier valor).

    --- CSS
    /* selector de atributo */
    /* dirección web que comience por http... */
    a[href^="http"] { 
      color: fuchsia;
      background-color: black;
    }
    
    /* dirección web que finalice por .org */
    a[href$=".org"] { 
      color: white;
      background-color: blue;
    }
    
    /* dirección web que contenga ...goog... */
    a[href*="joog"] { 
      color: black;
      background-color: yellow;
    }
    
    p[me_lo_he_inventado] {
      color:blueviolet
    }
      
    p[me_lo_he_inventado="True"] {
      color:green
    }
    
    p[me_lo_he_inventado="False"] {
      color:red
    }
    
    /* selecciona TODOS los atributos mi_attr */
    [mi_attr] {
      background-color: lightseagreen;
    }    

    --- HTML

    <a href="https://google.com" target="_blank"> Google</a>

    <p me_lo_he_inventado="True">Este párrafo tiene un atributo inventado por Yo :) -> True</p>
    <p me_lo_he_inventado="False">Este párrafo tiene un atributo inventado por Yo :) -> False</p>
    <p me_lo_he_inventado="no-sabe_no-contesta">Este párrafo tiene un atributo inventado por Yo :) -> Diferente a True y False</p>
  
    <p mi_attr="ok">Párrafo con mi_attr="ok"</p>
  
    <h4 mi_attr=""ok>Título H4 con mi_attr="ok"</h4>
