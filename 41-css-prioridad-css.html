Prioridades en CSS

Selectores del mismo tipo con propiedades en conflicto

La aplicación de CSS dependerá del orden en el archivo .css

Normalmente el último selector involucrado en aparecer en el .css será el aplicado

    ---CSS
    .fondo-gris {
      background-color: grey;
     }
     
     .fondo-amarillo {
       background-color: yellow;
     }

     --- HTML
     <div class="fondo-gris fondo-amarillo">
     <p>Lorem ipsum dolor sit amet consectetur, adipisicing elit. Modi ipsum quidem deleniti alias esse eaque inventore natus, quam ea beatae deserunt fugit aliquam repudiandae tenetur. Veritatis vero consequatur modi excepturi?</p>
     </div>

Selector de tipo vs selector de clase

Al ser el selector de clase más específico que el selector de tipo, se aplicará el estilo de la clase

    ---CSS
    div {
      color: blue;
    }
    
    p {
      color: black;
    }
    
    .texto-verde {
      color: green;
    }

    --- HTML
    <div>
    <p class="texto-verde">Lorem ipsum dolor sit amet consectetur, adipisicing elit. Modi ipsum quidem deleniti alias esse eaque inventore natus, quam ea beatae deserunt fugit aliquam repudiandae tenetur. Veritatis vero consequatur modi excepturi?</p>
    </div>

Selector de ID vs selector de clase

Al ser el selector de ID más específico que el selector de clase, se aplicará el estilo del ID

NOTA:

Es posible que se defina un selector de clase y un selector de ID con el mismo nombre

. 
    .fondo-gris {
      background-color: grey;
     }
     
     ...

     #fondo-gris {
       background-color: grey;
     }

No es muy recomendable seguir esta práctica!

    --- CSS
    .fondo-amarillo {
      background-color: yellow;
    }
    
    #bg-pink {
      background-color: pink;
    }

    --- HTML
    <div>
    <p id="bg-pink"" class="fondo-amarillo">Lorem ipsum dolor sit amet consectetur, adipisicing elit. Modi ipsum quidem deleniti alias esse eaque inventore natus, quam ea beatae deserunt fugit aliquam repudiandae tenetur. Veritatis vero consequatur modi excepturi?</p>
    </div>

Selector de ID vs selector de clase vs selector de atributo

Se aplicará el estilo de ID por su prioridad sobre la clase y el atributo

    --- CSS
    #google {
     color:  crimson;
    
    }
  
    .texto-azul{
     color: blue
    }
  
    [target="_blank"]{
     color: blueviolet;
    }

    --- HTML
    <div>
      <a id="google" class="texto-azul" href="http://google.com" target="_blank">Google</a>
    </div>
