Colores en CSS

Podemos referenciar el color mediante su nombre, un código hexadecimal o código RGB, como se explica a continuación.

  • Color:

    red, blue, green

  • Código Hexadecimal: #rrggbb o #rgb

    Valores válidos: 0-9 A-F

    Si los valores se repiten se pueden abreviar #22aaff = #2af

  • Código rgb + a(opcional)

    Valores válidos para rgb: 0...255

    Valores válidos para a (alpha o canal de transparencia): 0...1 siendo 0 totalmente transparente y 1 totalmente sólido, acepta valores intermedios

    rgb(255, 77, 25)

    rgba(255, 77, 25, 0.75)
