Formularios

¿Qué son los formularios?

Los formularios web son uno de los principales puntos de interacción entre un usuario y un sitio web o aplicación.

Los formularios permiten a los usuarios la introducción de datos, que generalmente se envían a un servidor web para su procesamiento y almacenamiento, o se usan en el lado del cliente para provocar de alguna manera una actualización inmediata de la interfaz (por ejemplo, se añade otro elemento a una lista, o se muestra u oculta una función de interfaz de usuario).

El HTML de un formulario web está compuesto por uno o más controles de formulario (a veces llamados widgets), además de algunos elementos adicionales que ayudan a estructurar el formulario general; a menudo se los conoce como formularios HTML.

Los controles pueden ser campos de texto de una o varias líneas, cajas desplegables, botones, casillas de verificación o botones de opción, y se crean principalmente con el elemento <input>, aunque hay algunos otros elementos que también hay que conocer.

Los controles de formulario también se pueden programar para forzar la introducción de formatos o valores específicos (validación de formulario), y se combinan con etiquetas de texto que describen su propósito para los usuarios con y sin discapacidad visual.

Formulario básico



<form>

El elemento HTML form <form> representa una sección de un documento que contiene controles interactivos que permiten a un usuario enviar información a un servidor web.

Atributos <form>

  • action
    Ubicación/url a dónde se envían los datos ingresados y validados en el formulario(Generalmente solicitan datos al usuario)
  • method
    Métodos HTTP: POST, GET, PUT, DELETE (generalmente los más usados son GET y POST)
  • target
    _blank, _parent, _self, _top (_blank abre una nueva pestaña del navegador)

<input>

El elemento HTML <input> se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario.

Atributos <input>

  • type
    Tipo de dato ingresado (en este caso texto) 
  • placeholder
    (marcador de posición) Texto quye se visualizará dentro de input para informar al usuario acerca del dato solicitado en ese campo 
  • required
    Establece como obligatorio este campo
