font-family: fuentes externas

fuentes de letra aplicadas desde una web externa o descargada

  --- CSS

  @font-face {
    font-family: 'pacifico';
    src: url(/assets/fonts/pacifico/Pacifico-Regular.ttf) format('truetype');
  }
  
  .handwriting-pacifico-greenyellow {
    font-family: pacifico;
    font-size: 20px;
    color: greenyellow;
    background-color: black;
  }
  
  .handwriting-pacifico-white {
    font-family: pacifico;
    font-size: 20px;
    color: white;
    background-color: black;
  }


  --- HTML

  <p style="font-family: pacifico;">Aplicado css inline</p>

  <p class="handwriting-pacifico-greenyellow">Aplicado desde CSS externo a la etiqueta <p></p>
  
  <p><span class="handwriting-pacifico-white">Aplicado desde CSS externo a la etiqueta <span></span></p>
