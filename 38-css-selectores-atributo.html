Selectores de atributos en CSS

  1. [attr]: Este selector selecciona todos los elementos HTML que tienen un atributo específico, independientemente de su valor. Es útil cuando deseas aplicar un estilo o una regla a todos los elementos que tengan ese atributo, sin importar qué valor tengan.

    2.     [data-toggle] {
      color: blue;
  }

    Situación real: Puedes utilizar este selector para aplicar un estilo a todos los elementos de un formulario que tienen el atributo required. Esto te permite destacar los campos obligatorios en un formulario web para que los usuarios sepan que deben completarlos.

  2. [attr=val]: Este selector selecciona todos los elementos HTML que tienen un atributo específico con un valor específico. Es útil cuando deseas aplicar un estilo o una regla solo a los elementos que coinciden exactamente con ese valor de atributo.

    3.       [href="https://www.ejemplo.com"] {
        text-decoration: underline;
    }

    Situación real: Cuando estás creando una página de inicio con enlaces a redes sociales, puedes usar este selector para aplicar un estilo específico solo a los enlaces que dirigen a Facebook.

  3. [attr~=val]: Este selector selecciona todos los elementos HTML que tienen un atributo específico donde el valor coincide con una palabra clave específica dentro de una lista separada por espacios. Es útil cuando deseas aplicar un estilo o una regla a elementos que tengan una palabra clave particular en su atributo.

    4.       --- CSS
      .etiquetas[data-color~="Rojo"] {
        color: red;
    }

    ---HTML
    <div class="etiquetas">Rojo Azul Verde</div>

    Situación real: Supongamos que tienes un sitio web que muestra artículos y cada artículo tiene etiquetas separadas por espacios en un atributo data-tags. Puedes usar este selector para aplicar estilos diferentes a los artículos que tengan una etiqueta específica.

  4. [attr|=val]: Este selector selecciona todos los elementos HTML que tienen un atributo específico con un valor que comienza exactamente con el valor especificado, seguido de un guion medio ("-"). Es útil cuando deseas seleccionar elementos que tienen un valor de atributo específico con un prefijo determinado.

    5.       --- CSS
      input[id|="lang"] {
        border: 1px solid gray;
    }

    ---HTML
    <input id="lang-es" type="text" />
    <input id="lang-en" type="text" />
    <input id="lang-fr" type="text" />

    Situación real: Supongamos que estás desarrollando un sitio web multilingüe y quieres aplicar estilos específicos a elementos con atributos lang que comienzan con un código de idioma en particular.

Ejemplo:

Había una vez en un pequeño pueblo rodeado de exuberantes bosques, un anciano llamado Don Manuel. Don Manuel vivía en una modesta casa de madera en lo profundo del bosque, donde pasaba sus días cuidando de un hermoso jardín de rosas. Sus rosales eran famosos en todo el lugar por su belleza y fragancia, y la gente viajaba desde lejos para contemplarlos.

Un día, una joven llamada Isabella llegó al pueblo en busca de refugio. Estaba perdida y cansada, y encontró la casa de Don Manuel mientras exploraba el bosque. Al ver el jardín de rosas, quedó maravillada por su belleza y decidió pedir ayuda a Don Manuel. Él la recibió con amabilidad y la invitó a quedarse en su casa hasta que encontrara su camino de regreso.

Con el paso de los días, Don Manuel y Isabella se hicieron amigos. Ella lo ayudaba en el jardín de rosas, y él le contaba historias de su juventud. A medida que pasaba el tiempo, el jardín de rosas parecía florecer aún más, como si las flores estuvieran agradecidas por la compañía de Isabella.

Finalmente, Isabella encontró el camino de regreso a su casa, pero decidió quedarse en el pueblo y seguir ayudando a Don Manuel con su jardín de rosas. Juntos, compartieron la belleza de las flores y la amistad, demostrando que a veces, en los lugares más inesperados, se pueden encontrar los tesoros más preciosos.

Los años pasaron, y la amistad entre Don Manuel y Isabella se convirtió en un lazo aún más profundo. El jardín de rosas continuó siendo el orgullo del pueblo, y su fragancia se extendió por toda la región. La gente decía que las rosas de Don Manuel y la sonrisa de Isabella iluminaban el lugar.
