Selectores agrupados en CSS

Utilizamos selectores agrupados cuando queremos aplicar un estilo en particular a más de un selector

    --- CSS
    h1,
h2,
h3{
  color: blue;
}

ol,
ul {
  font-style: italic;
}

.texto-blue {
  color: blue;
}

.texto-red {
  color: red;
}

.texto-green {
  color: green;
}
    --- HTML
    <h2>Título H2</h2>
    <h3>Título H3</h3>
    <ol class="texto-green">
      <li>Pan</li>
      <li>queso</li>
    </ol>
    <ul>
      <li>Vino tinto</li>
      <li>Vino blanco</li>
    </ul>
