  • Uno con tamaño y letra de <ul> 10px
  • Dos -> con tamaño 2em, es decir, dos veces el tamaño del contenedor, en este caso,<ul> 10px * 2 = 20px
  • Tres -> con tamaño 2rem, es decir, dos veces el tamaño del elemento raíz, en este caso,<html>

    Observa que el tamaño SI es el doble del tamaño del elemento raíz

Todos los elementos mostrados con este estilo son relativos a la etiqueta <html> y consecuentemente con el tamaño predeterminado de ese contenedor raíz(Root)
